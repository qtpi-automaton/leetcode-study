**Linked List Time Complexity (Worst Case):**

| Operation | Complexity |
|-----------|------------|
| Access by index | O(n) |
| Search | O(n) |
| Insert at head | O(1) |
| Insert at tail (with tail ptr) | O(1) |
| Insert at middle | O(n) |
| Delete at head | O(1) |
| Delete at tail/middle | O(n) |
